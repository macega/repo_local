updateDateTimeOffset(int, microsoft.sql.DateTimeOffset) (SQLServerResultSet)

This method was added in Microsoft SQL Server JDBC Driver 3.0.

Updates the value of the column specified to the DateTimeOffset Class value, given a zero-based column ordinal.

Syntax

public void updateDateTimeOffset(int index, microsoft.sql.DateTimeOffset x)

Parameters

index

The zero-based ordinal of a column.

x

A DateTimeOffset Class object.

Exceptions

SQLServerException

Remarks

You can retrieve a DateTimeOffset Class value with SQLServerResultSet.getDateTimeOffset.
